# Personal Website Prototype Design Prompt

## Design Style Reference
Style: Modern Dark Theme (inspired by designprompts.dev/modern-dark)
- Dark background with high contrast
- Clean, minimalist layout
- Modern typography
- Smooth transitions and subtle animations
- Professional and elegant aesthetic

## Website Structure

### 1. Header/Navigation Bar
**Components:**
- Personal name/logo (text-based, no profile image)
- Navigation menu with links:
  - About Me
  - Projects
  - Works Showcase
  - Skills
  - Contact
- Sticky navigation on scroll
- Responsive hamburger menu for mobile

### 2. Hero Section (招牌栏目)
**Content:**
- Large heading with personal name
- Tagline/Motto (座右铭)
- Brief self-introduction tagline
- Location indicator (座标)
- Call-to-action button (e.g., "View My Work" or "Get in Touch")
- No avatar/profile image - use placeholder or text-based design element

**Design Notes:**
- Full viewport height
- Center-aligned content
- Prominent typography
- Subtle background pattern or gradient

### 3. About Me Section (关于我)
**Content Subsections:**
- Introduction paragraph
  - Background story
  - Current role and expertise
  - Personal interests and hobbies
- Education background
  - University name: Hello, Huihui
  - Graduation year
  - Major/Field of study
- Career highlights
  - Key experiences
  - Professional achievements
  - Military service background (退役军人)

**Design Layout:**
- Two-column layout on desktop (text + timeline/cards)
- Single column on mobile
- Use cards or sections to organize information
- Timeline visualization for career/education history

### 4. Projects Section (项目展示)
**Content Structure:**
- Project categories:
  - University Projects (大学项目)
  - Personal Projects (个人项目)
  - Open Source Contributions
  - Collaborative Works
  
**Each Project Card Should Include:**
- Project title
- Brief description (技能栏分类的详细描述)
- Technologies used (技术栈)
- Project type indicator
- Link/button to view details or live demo
- NO project screenshots initially (placeholders only)

**Design Layout:**
- Grid layout (2-3 columns on desktop)
- Card-based design with hover effects
- Filter/category tabs at the top
- Responsive grid that adapts to mobile

### 5. Works Showcase Section (作品展示)
**Content Categories:**
- Media Works (媒体作品)
  - Video tutorials
  - Technical articles
  - Presentations
- Tutorial Series (教程方向)
  - Step-by-step guides
  - Code examples
  - Best practices
- Development Tools (开发小工具)
  - Utility tools
  - Plugins
  - Libraries

**Design Layout:**
- Masonry or grid layout
- Thumbnail placeholders (no actual images)
- Category filters
- Hover state with quick preview information
- Modal or separate page for detailed view

### 6. Skills Section (技能介绍)
**Content:**
- Technical Skills
  - Programming languages
  - Frameworks and libraries
  - Tools and platforms
- Professional Skills
  - Project management
  - Team collaboration
  - Communication

**Design Visualization:**
- Skill cards or tags
- Optional: visual indicators (bars, circles, or simple icons)
- Grouped by categories
- Clean, scannable layout

### 7. Tech Stack & Experience Section (技术分享)
**Content:**
- Cloud platforms experience
- Development environment setup
- Best practices and methodologies
- Blog posts or article links

**Design Layout:**
- Content cards or list format
- Links to external articles/posts
- Date stamps
- Reading time indicators

### 8. Dynamics/Updates Section (动态内容)
**Content Types:**
- Life updates (生活)
- Study notes (学问)
- Event participation (活动)
- Photography (照片分享) - placeholders only

**Design Layout:**
- Timeline or card-based feed
- Chronological order
- Filter by category
- Pagination or infinite scroll

### 9. Contact Section (交互版块)
**Contact Methods:**
- Email address
- GitHub profile link
- Pinterest link
- Zhihu (知乎) profile link
- WeChat QR code placeholder (NO actual QR image, just placeholder)
- Other social media links

**Design Elements:**
- Contact form with fields:
  - Name
  - Email
  - Message
  - Submit button
- Social media icon links (simple text links or icon placeholders)
- Contact information display

### 10. Footer (页尾)
**Content:**
- Copyright information
- Last updated date (改版日期)
- Quick navigation links
- Back to top button
- Optional: brief site map

**Design:**
- Dark background, slightly different from main sections
- Multiple columns on desktop
- Stacked layout on mobile
- Minimal and clean

## Global Design Specifications

### Color Palette (Modern Dark Theme)
- Primary background: Dark navy or charcoal (#0a0e27, #1a1a1a)
- Secondary background: Slightly lighter shade for cards/sections
- Primary text: Off-white or light gray (#e4e4e7, #f4f4f5)
- Accent color: Modern blue, purple, or teal for CTAs and highlights
- Border/divider: Subtle gray with low opacity

### Typography
- Headings: Modern sans-serif (e.g., Inter, SF Pro, or similar)
- Body text: Readable sans-serif with good line spacing
- Code snippets: Monospace font (e.g., Fira Code, JetBrains Mono)
- Font sizes: Clear hierarchy from H1 to body text

### Spacing & Layout
- Consistent padding and margins
- Section spacing: 80-120px vertical
- Content max-width: 1200-1400px
- Generous whitespace for readability

### Interactive Elements
- Smooth hover transitions (0.3s)
- Button states: normal, hover, active
- Link underlines or color changes on hover
- Subtle shadow effects for depth
- Smooth scroll behavior

### Responsive Breakpoints
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: < 768px

## Important Notes
1. **NO IMAGES**: Do not include actual photos, avatars, project screenshots, or QR codes. Use placeholder boxes with text labels instead.
2. **Text-based design**: Focus on typography, layout, spacing, and color to create visual interest.
3. **Wireframe clarity**: Ensure all sections are clearly labeled and structured.
4. **Modern dark aesthetic**: Maintain consistency with the dark theme throughout.
5. **Scalability**: Design should work well from mobile to large desktop screens.
6. **Accessibility**: Ensure sufficient contrast ratios for text readability.

## Additional Features to Consider
- Smooth scrolling navigation
- Section reveal animations on scroll
- Loading states for dynamic content
- Dark mode toggle (optional, since base is dark)
- Language switcher (EN/CN) if needed
- Search functionality for blog/articles
- Print-friendly styles
